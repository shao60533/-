# langgraph >=0.2 ships ``langgraph.prebuilt`` which TradingAgents
# imports at module load time; older releases lack the submodule and
# break ``from tradingagents.graph.trading_graph import TradingAgentsGraph``.
# Pin a known-compatible floor so Railway / fresh CI installs don't
# regress to a too-old transitive dep. See v1.0.2 P1-#1.

# v1.0.3 — pin langgraph-prebuilt explicitly. langgraph 1.x split
# ``langgraph.prebuilt`` into a SEPARATE package ``langgraph-prebuilt``;
# `pip install langgraph` no longer auto-pulls the submodule, so
# ``from langgraph.prebuilt import ToolNode`` (which TradingAgents
# uses at module load) fails with ModuleNotFoundError when only
# langgraph (without -prebuilt) is installed. Pin floor to a known-
# working version that ships ToolNode + the pieces TradingAgents
# imports.  ``test_tradingagents_import.py::test_langgraph_prebuilt_toolnode``
# guards future regressions.
langgraph-prebuilt>=1.0.9,<2
